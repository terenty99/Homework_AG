# Алгебра на графах: BFS и MST

---

## Задача 1: Обобщенный BFS (Поиск в ширину)

Реализует поиск в ширину через **обобщенное умножение вектора (фронта) на матрицу смежности**. 

В алгоритме используется процедура вычисления 3 сценариев:

1. **Сложение (`+ := min`)**: 
   Если в одну и ту же вершину можно прийти по нескольким путям, нам нужен самый короткий. Поэтому вместо обычного сложения мы просто выбираем минимальное значение.

2. **Умножение (`* := f * m = f + 1`)**: 
   Когда мы переходим из текущей вершины (где пройдено расстояние `f`) по ребру графа, наш путь становится длиннее на один шаг. Поэтому мы просто прибавляем `1` к текущему расстоянию `f`.

3. **Маска (`mask`)**: 
   Чтобы алгоритм не ходил по кругу и не возвращался в те вершины, где мы уже были, применяется маска. Если мы видим, что расстояние до вершины уже найдено (значение не равно `None`), то в новом фронте мы принудительно заменяем значение этой вершины на `None`. 

Алгоритм выводит процесс умножения вектора на матрицу шаг за шагом, показывая найденный фронт (`f=`) и глобальные расстояния (`n=`).

## Задача 2: Обобщенное замыкание графа (Поиск MST)

Реализует поиск Минимального остовного дерева (MST) через **обобщенное умножение матрицы на матрицу**.

Мы возводим матрицу смежности в степень (умножаем саму на себя) до тех пор, пока она не перестанет меняться. Правила для математики здесь следующие:

1. **Умножение (`* := max`)**: 
   Когда мы идем по пути, состоящему из нескольких ребер, нас интересует только самое "тяжелое" (максимальное) ребро на этом пути.
2. **Сложение (`+ := min`)**: 
   Из всех возможных путей между двумя вершинами алгоритм выбирает тот, где это самое "тяжелое" ребро будет как можно меньше (минимальным).

**Как получается ответ:**
Когда матрица перестает меняться (достигается замыкание), мы получаем "матрицу узких мест". Чтобы узнать, какие именно ребра образуют MST, мы просто сравниваем исходную матрицу с итоговой:
Если вес ребра в исходной матрице совпадает с весом в замкнутой матрице — это ребро является частью Минимального остовного дерева.

## Как запустить

Вам потребуется установленный **Python версии 3.10** и библиотека `numpy`:

```bash
pip install numpy
```

Для запуска выполните:
```bash
python main.py
```

После запуска в консоли появится меню, где нужно будет ввести `1` или `2` для выбора соответствующей задачи.
